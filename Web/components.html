BUTTON

button 태그와 .btn 클래스명에 style 입혀져있음.
width값은 클래스명을 설정하지 않으면 양쪽 24px의 간격으로 기본 설정되어있으며,
.full은 width 100% / .half는 width 50% 등으로 추가하여 사용할 수 있습니다.

height값은 설정하지 않으면 40px으로 설정되었으며, big 50px / small 30px으로 되어있습니다.
